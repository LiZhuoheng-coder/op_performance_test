Seagull 算子性能测试
========

## build
- 修改 CMakeLists.txt 中 `set(CMAKE_CROSSCOMPILING TRUE)` 为 `set(CMAKE_CROSSCOMPILING False)`
```sh
mkdir build && cd build
cmake ..
ninja # or make -j4
```
## 运行

- 直接运行build目录中的 `./seagull_op_test`
